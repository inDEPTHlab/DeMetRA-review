## DeMetRA - the Developmental Methylation Risk Atlas 

This repo hosts the database and code for an interactive application that allows users to explore the existing literature on Methylation Profile Scores (MPS) developed or applied in pediatric / developmental samples. 

The app is hosted [here](https://indepthlab-demetra.share.connect.posit.cloud).

It was originally developed to accompany this systematic literature review: 
***Methylation profile scores in early life: a systematic review and developmental risk atlas*** [Pre-print](http://dx.doi.org/10.2139/ssrn.5852502)

It is meant as a living resource, where authors can upload their own work using the Submission form.
