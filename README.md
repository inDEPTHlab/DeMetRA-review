**DeMetRA** is an interactive database and web application for exploring the existing literature on **Methylation Profile Scores (MPS)** developed or applied in pediatric / developmental samples.

DeMetRA is a **living resource**, publicly hosted at:
**[indepthlab-demetra.share.connect.posit.cloud](https://indepthlab-demetra.share.connect.posit.cloud)**

Follow the link to explore exiting MPSs or submit your own work using the in-app submission form.

It was originally built by the [inDEPTH lab](https://charlottececil.com/) at Erasmus MC, to accompany the systematic review:

> ***Methylation profile scores in early life: a systematic review and developmental risk atlas***
> → [Read the pre-print](http://dx.doi.org/10.2139/ssrn.5852502)
